WSGI&UWSGI（不考虑uwsgi协议）

  1. WSGI：全称是Web Server Gateway Interface，WSGI不是服务器，python模块，框架，API或者任何软件，只是一种规范，描述web server如何与web application通信的规范。server和application的规范在PEP3333中有具体描述。要实现WSGI协议，必须同时实现web server和web application，当前运行在WSGI协议之上的web框架有Bottle, Flask, Django。

  2. uWSGI：是一个web服务器，实现了WSGI协议、uwsgi协议、http协议等。
